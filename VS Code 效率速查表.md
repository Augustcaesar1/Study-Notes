# VS Code 效率速查表

### 一、 命令行与基础控制 (CLI & UI)

| **快捷键/命令**        | **功能**                       | **备注**                                                   |
| ---------------------- | ------------------------------ | ---------------------------------------------------------- |
| `code .`               | **当前目录打开**               | 在终端输入，用 VS Code 打开当前文件夹（最常用）            |
| `code <路径>`          | 打开指定路径                   | 你的笔记项                                                 |
| `Ctrl` + ```           | **打开/隐藏终端**              | 你的笔记项，Terminal 面板                                  |
| `Ctrl` + `B`           | 显示/隐藏侧边栏                | 增加可视面积                                               |
| `Ctrl` + `Shift` + `P` | **命令面板 (Command Palette)** | **修正：** 这里是输入所有命令的地方（如 `>Reload Window`） |
| `Ctrl` + `P`           | **快速打开文件 (Go to File)**  | **修正：** 你的笔记归类有误，这个通常用于按文件名搜文件    |

### 二、 光标移动与跳转 (Navigation)

| **快捷键**              | **功能**              | **备注**          |
| ----------------------- | --------------------- | ----------------- |
| `Ctrl` + `G`            | **跳转到行**          | 输入 `:行号` 跳转 |
| `Ctrl` + `右箭头`       | 光标按单词向右跳      | 英文环境非常有用  |
| `Ctrl` + `左箭头`       | 光标按单词向左跳      | **补全**          |
| `Home` / `End`          | 跳转到行首 / 行尾     | **补全**          |
| `Ctrl` + `Home` / `End` | 跳转到文件头 / 文件尾 | **补全**          |

### 三、 文本编辑与行操作 (Editing)

| **快捷键**                | **功能**                | **备注**                                                     |
| ------------------------- | ----------------------- | ------------------------------------------------------------ |
| `Shift` + `下箭头`        | 选中当前行/向下选中     | 你的笔记项                                                   |
| `Alt` + `上/下箭头`       | **移动当前行**          | 代码搬运神技，无需复制粘贴                                   |
| `Shift` + `Alt` + `上/下` | **复制当前行**          | **补全：** 快速克隆当前行到上/下方                           |
| `Ctrl` + `/`              | **注释/取消注释**       | **补全：** 必会快捷键                                        |
| `Shift` + `Alt` + `F`     | **格式化代码 (Format)** | **修正：** 标准默认键位。`Ctrl+Shift+I` 通常是浏览器控制台或用户自定义 |
| `Ctrl` + `Z` / `Y`        | 撤销 / 重做             | **补全**                                                     |

### 四、 “大杀器”：多光标与重构 (Multi-Cursor & Refactor)

这是 VS Code 最强大的功能区，学会后效率翻倍。

| **快捷键**            | **功能**                 | **场景描述**                                                 |
| --------------------- | ------------------------ | ------------------------------------------------------------ |
| `Ctrl` + `D`          | **选中下一个相同词**     | **神技**。选中一个变量名，按几次 `Ctrl+D`，可以同时修改这几个地方。 |
| `Alt` + `鼠标左键`    | **多处光标点击**         | 在不同位置点一下，可以同时在多处打字。                       |
| `Shift` + `Alt` + `I` | 在所有选中的行尾插入光标 | 选中多行文本，按下此键，光标会出现在每一行的末尾。           |
| `F2`                  | **重命名符号**           | 选中变量名按 F2，智能修改整个项目中所有引用该变量的地方（比查找替换更安全）。 |

### 五、 搜索 (Search)

| **快捷键**             | **功能**                      |
| ---------------------- | ----------------------------- |
| `Ctrl` + `F`           | 当前文件查找                  |
| `Ctrl` + `H`           | 当前文件替换                  |
| `Ctrl` + `Shift` + `F` | **全局查找** (在整个项目中找) |

------

### 建议操作

1. **区分 `Ctrl+P` 和 `Ctrl+Shift+P`：**
   - 找**文件**用 `Ctrl + P`。
   - 找**命令**（比如设置主题、重载窗口、Git命令）用 `Ctrl + Shift + P`。
2. **配置格式化：** 建议在设置中开启 **"Format On Save"**（保存时自动格式化），这样你就不需要手动按快捷键整理代码了。
